Request Recipes

Fetch Insights

Get KPI and distribution payload for a user.

curl -X GET "{{ request.host_url.rstrip('/') }}/api/insights?user_id=1"

Fetch Timeline

Get timeline buckets for upcoming deadlines.

curl -X GET "{{ request.host_url.rstrip('/') }}/api/timeline?user_id=1&days=14"

Export CSV

Export tasks for reporting and offline analysis.

curl -X GET "{{ request.host_url.rstrip('/') }}/exportTasks?user_id=1&user_name=demo" -o tasks.csv
