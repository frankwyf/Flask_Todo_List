Portfolio API Guide

This page documents major route contracts for demo and interview walkthrough.

Authentication

  • GET /: Open login page.
  • POST /newAccount: Register new user. Form fields: name, mailenter, psw.
  • POST /doList: Login and open user dashboard. Form fields: name, psw.

Task Management

  • POST /createAss: Create a task.
  • GET /completeTask: Mark task complete.
  • GET /undoTask: Undo completion status.
  • GET /deleteTask: Delete one task.
  • POST /subEdit: Submit task edit.

Search and Analytics

  • POST /searchStatus, /searchmName, /searchaName
  • GET /searchOverdue
  • GET /visualCompletion, /visualPri, /visualModule
  • GET /api/summary?user_id=<id>: Dashboard summary metrics.
  • GET /api/insights?user_id=<id>: KPI + priority/module distribution snapshot.
  • GET /api/timeline?user_id=<id>&days=14: Deadline timeline aggregation.
  • GET /api/tasks?user_id=<id>: User task list as JSON.

Portfolio Support

  • GET /portfolio/api: This documentation page.
  • GET /portfolio/architecture: System architecture diagram.
